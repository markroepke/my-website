Introducing flxr - an R package to retrieve movie data

I am happy to say that I’ve created my first R package: flxr. This package is designed to scrape basic data about movies.

It can be installed using the devtools package:

devtools::install_github("markroepke/flxr")
library(flxr)

The flxr package retrieves information about movies using only a user-provided movie title. For example, if we wanted to find the director of The Godfather, we could use the flx_mov_director() function:

flx_mov_director(movie = "The Godfather")
## Returning movie director for The Godfather:
## [1] "Francis Ford Coppola"

You will notice the message above specifying that the director for The Godfather is being returned. This is printed to ensure the user that the correct movie was found based on the movie name provided to the function.

There are many other functions available in flxr to retrieve other data. These functions include:

  • flx_mov_desc() to return the movie description
  • flx_mov_genre() to return the movie genre
  • flx_mov_runtime() to return the movie runtime
  • flx_mov_user_rtg() to return the movie average IMDB user rating
  • flx_mov_mc_rtg() to return the movie Metacritic rating
  • flx_mov_lead_cast() to return the movie title cast


And finally, there is a function to return all of the above information:

library(magrittr)
flx_mov_details(movie = "The Godfather") %>%
  knitr::kable()
movie description genre director lead_cast runtime user_rating metacritic_rating
The Godfather The aging patriarch of an organized crime dynasty transfers control of his clandestine empire to his reluctant son. Crime Francis Ford Coppola Marlon Brando, Al Pacino, James Caan 175 9.2 100


I am hoping to use this package in the coming days to look into the Oscar-nominated films.
